# Testing

This directory contains the testing and validation utilities used during development.

Contents:

- `validate_submission.py` – Validates the generated submission against the competition format.
- `generate_edge_cases.py` – Creates synthetic edge-case candidates for robustness testing.
- `Garuda_Testing_and_Validation.ipynb` – End-to-end testing notebook demonstrating the complete validation workflow.

These tools are intended for local verification and are not required for generating the final submission.
